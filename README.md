# 💪 AI Gym Planner
#### 🔰Overview

- Gym Planner is a full-stack web application designed to help users create and manage personalized workout routines with ease. The platform allows users to sign up, define their fitness goals, and generate structured training plans tailored to their needs. With a responsive and user-friendly interface, it ensures a smooth experience across devices.

- The application is built using a modern TypeScript-based stack, combining a fast React frontend with a scalable Express backend and a PostgreSQL database powered by Prisma. It is deployed on the cloud for seamless accessibility and performance.

- Gym Planner focuses on simplicity, efficiency, and flexibility, making it a practical tool for anyone looking to organize and track their fitness journey.

## 🛠️ Tech Stack

### 🎨 Frontend
- React.js + TypeScript
- Vite (for fast development and build)
- Tailwind CSS (for styling and responsive UI)
### ⚙️ Backend
- Node.js
- Express.js
- TypeScript
### 🗄️ Database & ORM
- PostgreSQL (hosted on Neon)
- Prisma ORM
### 🔐 Authentication
- Neon Auth (frontend-based authentication)
### ☁️ Deployment
- Render (for both frontend and backend hosting)


## 🎯 Project Purpose

- The purpose of Gym Planner is to provide a simple and efficient platform for users to organize and manage their fitness routines. Many people struggle with planning structured workouts or staying consistent with their fitness goals. This project aims to solve that by offering a personalized and easy-to-use solution for creating and tracking workout plans.

- It also serves as a full-stack development project to demonstrate the integration of modern web technologies, including a TypeScript-based frontend and backend, database management with Prisma, and cloud deployment. The application focuses on delivering a smooth user experience while maintaining scalability and clean architecture.
